Project Info

ZContacts is a Customer Relationship Management (CRM) application designed to help you manage your contacts efficiently. This project utilizes modern web technologies to provide a smooth and responsive user experience.

Technology Stack

  • Tailwind CSS: A utility-first CSS framework for styling the application.
  • Flowbite: A library of interactive components built on Tailwind CSS.
  • Feather Icons: Simple and elegant icons for UI elements.
  • SweetAlert2: Beautiful and customizable alert boxes for user notifications.
  • OpenStreetMap: A collaborative project to create a free editable map of the world, used for displaying contact locations.

GitHub Repository

ZContacts is authored by Zacky Achmad. The source code is available on GitHub.
